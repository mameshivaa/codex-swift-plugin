What the compiler doesn't catch

Missing Permissions

Camera, photos, location, contacts, microphone, Bluetooth, HealthKit, Face ID, speech, tracking... Code compiles. App crashes on first API call.

Bundle.module Misuse

Image("x") in SPM packages loads from wrong bundle. Assets silently return nil. No compiler warning.
